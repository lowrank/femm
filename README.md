# femm (Finite Element Method in MATLAB for 2D simulation)

- [x] Mesh for 2D (triangle),
- [x] FunctionSpace class.
- [x] Function method.
- [x] Quadrature modes.
- [x] Mesh partition.
- [ ] Bilinear Form.
- [ ] Linear Form.

# Pre-build

libraries needed: ``trianlge``, ``metis``(5.x).

- ``apt-get install libtriangle-dev``
- ``apt-get install libmetis-dev``

# Build
- ``make``
